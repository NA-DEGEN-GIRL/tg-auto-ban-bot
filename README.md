# Telegram 자동 강제퇴장 & 스팸/욕설/도배 방지 봇

텔레그램 그룹/공지방(채널/토론방 등) 관리용으로 만든 자동화 필터 봇입니다.

---

## 주요 기능

### 메인 봇 (`ban_bot.py`)
- **자동 강퇴:**  
  새로 들어온 사용자(사람 멤버)는 자동으로 그룹에서 즉시 강퇴(언밴 포함, 재초대 가능).  
  *봇 계정은 강퇴하지 않음. `auto_kick` 설정으로 활성화/비활성화 가능*
- **러시아어(키릴 문자) 자동 영구 차단:**  
  닉네임/유저네임 **또는 메시지 본문**에 키릴 문자가 감지되면 영구 강퇴 (unban 없음). 메시지의 경우 삭제 후 모든 공지채널·현재방에서 영구 ban
- **username 없는 계정 자동 영구 차단:**  
  username(@아이디)이 없는 신규 입장자는 영구 강퇴 (unban 없음). *키릴/username 영구밴은 `KICK_EXCEPTIONS` 방에서도 적용*
- **조용한 모드(`SILENT_CHAT_ID`):**  
  지정한 방에서는 삭제·강퇴는 그대로 수행하되 안내 메시지는 보내지 않고 로그만 남김
- **금칙어 실시간 설정(`BADWORDS`):**  
  `config.json`의 `BADWORDS`를 편집하면 봇 재시작 없이 즉시 반영. 영문은 대소문자 무시
- **스팸 링크, 홍보, 봇/claim 패턴 완벽 차단:**  
  - 첫 댓글에 http/https, 도메인, 하이퍼링크, 숨겨진 링크(`text_link`), `@xxxbot`, `claim` 등  
    "모든 형태의 홍보/스팸/봇 마케팅"이 있으면 즉시 강퇴 및 메시지 삭제.
  - 기존 유저의 스팸/링크 댓글은 해당 댓글만 즉시 삭제(강퇴 X).
- **욕설/특정 키워드(변형 포함) 필터:**  
  - BADWORDS 리스트에 등록된 단어가 변형되어 있어도 탐지 및 삭제.
- **도배/장문 차단:**  
  - 텍스트/캡션이 1000자 이상인 메시지는 자동으로 삭제.
- **첫 댓글 환영 기능:**  
  - 링크·욕설·도배에 안 걸리는 첫 댓글일 경우 환영 메시지 자동 전송.

### 구독자 목록 수집 (`get_subscribers.py`)
- **Telethon(Client API)을 이용한 채널 구독자 목록 수집**
- 다양한 검색 쿼리(키릴/라틴/숫자)로 최대한 많은 구독자 수집
- 러시아어 닉네임 자동 분류 및 JSON 저장

### 일괄 강퇴 (`ban_from_list.py`)
- **수집된 JSON 파일 기반 러시아인들 일괄 강퇴**
- Bot API를 이용하여 `cyrillic_users.json`에 있는 유저들을 자동 강퇴

## 설정

### 1. 메인 봇 설정 (`config.json`)

`copy.config.json`을 `config.json`으로 복사 후 수정:

```json
{
    "TOKEN": "YOUR_BOT_TOKEN",
    "ADMIN_ID": 123456789,
    "NOTICE_CHAT_ID": [-1001234567890],
    "KICK_EXCEPTIONS": [-1009876543210],
    "SILENT_CHAT_ID": [-1001112223334],
    "BADWORDS": ["김대중", "운지", "섹스", "nice"]
}
```

| 키 | 필수 | 설명 |
|---|---|---|
| `TOKEN` | ✅ | BotFather에서 발급받은 봇 토큰 |
| `ADMIN_ID` | ✅ | 관리자 텔레그램 ID (숫자). 이 계정의 메시지는 필터에서 제외 |
| `NOTICE_CHAT_ID` | ✅ | 감시할 공지 **채널** ID 목록 (배열) |
| `KICK_EXCEPTIONS` | ⬜ | 신규입장 **일시강퇴를 면제**할 채팅 ID 목록. 단 영구밴(러시아어/username없음)은 여기서도 그대로 적용됨 |
| `SILENT_CHAT_ID` | ⬜ | **안내 메시지를 보내지 않을** 채팅 ID 목록. 삭제·강퇴 같은 조치는 그대로 하고 로그만 남김 |
| `BADWORDS` | ⬜ | 금칙어 배열. **봇 재시작 없이 실시간 반영**(파일 저장 즉시). 영문은 대소문자 무시. 생략하면 코드의 기본 목록 사용 |

### 2. 구독자 수집/일괄 강퇴 설정 (`.env`)

```env
# Telegram Client API (my.telegram.org에서 발급)
API_ID=12345678
API_HASH=your_api_hash_here

# 본인 전화번호 (국가코드 포함)
PHONE=+821012345678

# 채널 username (@없이) 또는 채널 ID
CHANNEL=your_channel_username

# Bot API (일괄 강퇴용)
BOT_TOKEN=your_bot_token_here
```

#### API_ID / API_HASH 발급 방법
1. https://my.telegram.org 접속
2. 전화번호로 로그인
3. **API development tools** 클릭
4. `api_id`와 `api_hash` 복사

## 사용법

### 1. 메인 봇 실행

```bash
python ban_bot.py
```

**최초 인증:** 텔레그램에서 관리자 계정으로 `/auth` 명령 전송

#### 명령어

| 명령어 | 설명 |
|--------|------|
| `/auth` | 최초 관리자 인증 (1회) |
| `/stop` | 봇 일시정지 |
| `/restart` | 봇 재시작 |

### 2. 채널 구독자 목록 수집

```bash
python get_subscribers.py
```

**첫 실행 시:** 콘솔에서 텔레그램 인증 코드 입력 필요

**출력 파일:**
- `all_subscribers.json` - 전체 구독자 목록
- `cyrillic_users.json` - 러시아어 닉네임 유저만

### 3. 일괄 강퇴 실행

```bash
python ban_from_list.py
```

`cyrillic_users.json`에 있는 유저들을 순차적으로 강퇴합니다.

## 동작 흐름

### 1) 새 유저 입장 시 (`kick_user`)

```
새 유저 입장
   │
   ├─ 봇 계정?                          ──▶ 무시 (아무 조치 안 함)
   │
   ├─ 닉네임·username에 키릴문자(러시아어)? ──▶ 🔴 영구 강퇴 (ban, unban 없음)   ※예외방에서도 적용
   │
   ├─ username 없음?                     ──▶ 🔴 영구 강퇴 (ban, unban 없음)   ※예외방에서도 적용
   │
   ├─ KICK_EXCEPTIONS 방?                ──▶ 통과 (강퇴 안 함)
   │
   └─ 그 외                              ──▶ 🟡 일시 강퇴 (ban + unban, 재초대 가능)
                                             └ 강퇴 성공 & silent 방 아니면 → "자동 강퇴되었습니다" 안내
                                               (실패하면 안내 없이 로그만, 다음 유저 계속 처리)
```

### 2) 댓글·메시지 처리 (`spam_reply_handler`)

```
메시지 수신
   │
   ├─ 시스템(777000)·채널 게시글?         ──▶ 무시
   ├─ 미인증/정지/관리자(ADMIN)?          ──▶ 무시
   │
   ├─ 작성자 기록 (첫 댓글인지 판정)
   │
   ├─ 텍스트·캡션 없음(스티커 등)?         ──▶ 무시 (미디어는 그룹 설정에서 차단)
   ├─ 메시지에 키릴문자(러시아어)?          ──▶ 🔴 삭제 + 모든 공지채널·현재방 영구 ban
   ├─ 답글 아닌 공지채널 글?              ──▶ 무시
   │
   ├─ 링크/홍보(@봇·claim) 포함?
   │     ├─ 없음 + 첫 댓글 + 욕설X + 도배X  ──▶ 환영 인사 후 종료 (silent 방이면 인사 생략)
   │     └─ 있음 ──────────────────────────────┐
   │                                           ▼
   ├─ [링크 삭제]  첫댓글 또는 공지답글 + 링크  ──▶ 메시지 삭제
   │
   ├─ [강퇴]      첫댓글 + 링크                ──▶ 🔴 모든 공지채널 + 현재방 영구 ban (재구독 불가)
   │
   ├─ [욕설]      금칙어 포함                  ──▶ 메시지 삭제 (+ "금지어 삭제" 안내 · silent면 생략)
   │
   └─ [도배]      1000자 이상                 ──▶ 메시지 삭제 (+ "장문 도배 삭제" 안내 · silent면 생략)
```
> 첫 댓글이라도 **욕설·도배는 삭제**됩니다(이때 강퇴는 안 함). 강퇴는 첫 댓글에 **링크**가 있을 때만.

### 강퇴(ban) 종류 한눈에 보기

| 트리거 | 조치 | unban | 재가입·재구독 | 예외방 면제 | 안내 메시지 |
|---|---|:---:|:---:|:---:|---|
| 키릴(러시아어) 닉네임 — 입장 | 영구 ban | ✗ | ❌ 불가 | ❌ (항상 강퇴) | 없음 |
| username 없음 — 입장 | 영구 ban | ✗ | ❌ 불가 | ❌ (항상 강퇴) | 없음 |
| 일반 입장 | 일시 ban+unban | ✓ | ✅ 가능 | ✅ 면제됨 | "자동 강퇴되었습니다" (silent 생략) |
| 첫 댓글 + 링크 | 영구 ban (전 공지채널+현재방) | ✗ | ❌ 불가 | — | 없음 |
| 메시지에 키릴(러시아어) 포함 | 영구 ban (전 공지채널+현재방) | ✗ | ❌ 불가 | — | 없음 (메시지 삭제) |

### 메시지(삭제·안내) 한눈에 보기

| 상황 | 메시지 조치 | 안내 문구 | silent 방 |
|---|---|---|---|
| 깨끗한 첫 댓글 | — | "첫 댓글 고맙다 🎉" | 인사 생략 |
| 링크 댓글 (첫댓글/공지답글) | 삭제 | 없음 | — |
| 욕설(금칙어) | 삭제 | "○○ 금지어 삭제" | 안내 생략 |
| 도배(1000자 이상) | 삭제 | "○○ 장문 도배 삭제" | 안내 생략 |

### 특수 설정 요약

- **`KICK_EXCEPTIONS`** — 신규 입장 *일시강퇴만* 면제. 영구밴(러시아어·username없음)과 댓글 필터는 그대로 동작.
- **`SILENT_CHAT_ID`** — 삭제·강퇴 등 *조치는 그대로*, **안내 메시지만** 생략(로그만 남김).
- **`BADWORDS`** — `config.json`에서 실시간 편집(재시작 불필요), 영문 대소문자 무시, 부분일치(예: `이기` → `이기적`도 탐지).

### 일괄 강퇴 흐름
```
1. get_subscribers.py 실행 → cyrillic_users.json 생성
2. ban_from_list.py 실행 → JSON 기반 일괄 강퇴
```

## 주의사항

### 보안
- `.env`, `*.session`, `config.json`은 절대 공유 금지
- `.gitignore`에 아래 항목 추가 권장:
  ```gitignore
  .env
  *.session
  *.session-journal
  config.json
  all_subscribers.json
  cyrillic_users.json
  writers.json
  auth_status.json
  ```

### 권한
- 봇은 채널/그룹에서 **관리자** 권한 필요 (특히 "사용자 차단" 권한)
- `get_subscribers.py`는 **채널 관리자 계정**으로 로그인해야 구독자 조회 가능

### 제한사항
- 채널 구독자 목록은 API 제한으로 **100% 수집 불가** (최대 ~1000-2000명)
- 이후 새 가입자는 `ban_bot.py`의 실시간 감시로 차단
